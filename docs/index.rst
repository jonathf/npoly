NumPoly -- Numerical Polynonmials
=================================

.. image:: .static/numpoly_logo.svg
   :height: 200 px
   :width: 200 px
   :align: center

Numpoly is a generic library for creating, manipulating and evaluating
arrays of polynomials based on ``numpy.ndarray`` objects.
